NAME

Bugzilla::DB::Oracle - Bugzilla database compatibility layer for Oracle

DESCRIPTION

This module overrides methods of the Bugzilla::DB module with Oracle specific implementation. It is instantiated by the Bugzilla::DB module and should never be used directly.

For interface details see Bugzilla::DB and DBI.
